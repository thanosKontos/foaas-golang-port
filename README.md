# foaas-golang
A port of the infamous foaas in Golang

# Installation

* Install glide: https://github.com/Masterminds/glide#install
* `cd $GOPATH`
* `mkdir -p thanosKontos`
* `cd thanosKontos`
* `clone git@github.com:thanosKontos/foaas-golang-port.git`
* `cd foaas-golang-port`
* `glide install`
* `go run main.go`
